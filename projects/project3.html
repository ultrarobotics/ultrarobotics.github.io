This project was the capstone of my first year, integrating multiple disciplines to create a robot capable of scanning store shelves and removing misplaced items. I led the software architecture and motion planning aspects of the project.

System Architecture
  • Developed software layout for ROS so that all nodes could integrate seamlessly.
  • Collaborated with team members (Vision, Navigation, Human-robot-interaction) to ensure compatibility between all produced software.
Motion Planning
  • Used MoveIt for robotic arm positioning.
  • Created movement sequences for picking and placing products, and greeting customers.
Validation
  • Validated the motion in both simulated and real environments, ensuring reliability in a store-like setting.
